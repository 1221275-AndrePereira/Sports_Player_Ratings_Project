# Sports_Player_Ratings_Project

In this project i used C# for the Backend; HTML, CSS, TypeScript and Angular for the Frontend.
The data used in this project was obtained via all the games featured here.

## What's in it

Player ratings for every season(videogame) I could possibly find.

## How to run

For now, you need to run the backend by clicking CTRL+F5 and choosing more C# options and then WebApi(https).
Open a cmd terminal in Visual Studio Code and run the command "cd Frontend_App" and then the command "ng serve --open".
